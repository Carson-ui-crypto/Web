## ✨ 核心亮點
- **原生 App 級別動畫**：採用 `cubic-bezier` 高性能過渡曲線，打造點擊排行時「由右往左絲滑進場、點擊返回向右退場」的流暢體驗。
- **無縫邊界嵌死**：側邊欄經極致邊距歸零（Margin/Padding 0），完美貼實手機殼左側與底部。
- **階梯式視覺層次**：前三名維持尊貴大封面與 Impact 字體，四名以後自動切換為緊湊微縮版。
- **極簡選單設計**：移除所有臃腫的圓形外框，點擊「最新更新」展開一排排文字選單，選中時自動在右側彈出「✓」剔號。
- **前端代碼防護**：JS/CSS 全面實施壓縮與加密混淆，並導入「防右鍵、防 F12」安全機制。
